# Netflix Data Analysis 📊

## 📌 Project Overview

In this project, I performed data analysis on a Netflix dataset using Python. The goal was to explore content distribution and extract meaningful insights.

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📊 Analysis Performed

* Data cleaning and handling missing values
* Analysis of content types (Movies vs TV Shows)
* Country-based content distribution
* Yearly content trend analysis

## 🔍 Key Insights

* Movies are more common than TV Shows on Netflix
* The United States produces the highest amount of content
* There has been a significant increase in content production in recent years

## 📁 Files

* `proje.ipynb` → main analysis notebook
* `netflix_titles.csv` → dataset

## 🚀 What I Learned

* Data analysis workflow
* Data visualization
* Handling real-world datasets

  ## 📈 Sample Insights

* The majority of content on Netflix consists of movies rather than TV shows, indicating a stronger focus on film distribution.
* The United States dominates content production, suggesting its central role in the platform’s content strategy.
* A significant increase in content production is observed after 2015, reflecting Netflix’s global expansion and investment in original content.
